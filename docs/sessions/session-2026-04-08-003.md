## Issues & Technical Debt

1. **Dual summary storage uses SQLite, not JSONL** ‚Äî Intentional design deviation. Summaries in `summaries` table rather than `index/summaries.jsonl`. Better collocated with index data. REQUIREMENTS.md not updated to reflect this.

2. **`_parse_frontmatter` is a simple custom parser** ‚Äî `src/assistonauts/archivist/service.py:23-39` only handles simple scalar YAML values. Multi-value fields (lists, nested) won't parse. Using `pyyaml` would be more robust. Non-blocking since all current frontmatter uses simple scalars.

3. **Graph analyzer uses substring matching for slugs** ‚Äî `src/assistonauts/tools/curator.py:96` uses `if target in article` which is substring match. A link target "net" would match "neural-networks.md". Should use exact slug-to-stem matching.

4. **No contract tests for Phase 3** ‚Äî All tests are unit tests. No contract tests with recorded LLM fixtures for the Curator agent.

5. **FTS5 operator words not handled** ‚Äî Query sanitization strips punctuation but doesn't escape FTS5 boolean operators (NOT, AND, OR).
