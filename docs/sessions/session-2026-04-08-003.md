## Session Notes

- sqlite-vec v0.1.9 works on aarch64 Linux without issues.
- Multimodal image messages use the standard OpenAI/litellm content blocks format (`[{"type": "image_url", "image_url": {"url": "data:..."}}]`).
- LLM cache eviction uses `SUM(LENGTH(content))` not `PRAGMA page_count` because SQLite doesn't reclaim pages after DELETE without VACUUM.
- Curator singleton uses `threading.Lock` — adequate for CLI. Phase 5 scaling may need process-level enforcement.
- FakeEmbeddingClient consolidated in `tests/helpers.py` alongside FakeLLMClient.
- `numpy` was removed from dependencies — embedding averaging uses plain Python lists.
- The `assistonauts index` command attempts to create an embedding client from workspace config. Falls back to FTS-only if no embedding model is configured.
